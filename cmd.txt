6. Создать файл new.json - touch new.json
7. Добавить под гит - git add new.json
8. Закоммитить файл - git commit -m "add new file"
9. Отправить файл на внешний репозиторий - git push
10. Отредактировать содержимое файла - vim new.json
11. Отправить на внешинй репозиторий - git add new.json; git push

13. В файл preferences.json добавить информацию - vim preferences.json
